# Reports

This directory contains the reports and plots generated from the experiments.

## Contents

-   `Report.pdf`: The final project report in PDF format.
-   `Latex_report/`: The LaTeX source files for the project report.
-   `plots/`: This directory contains various plots generated during the analysis and model evaluation. The plots visualize the model's performance, such as cumulative reward, regret, and arm selection frequency.
-   `recommendations/`: This directory contains the recommendations generated by the model.

## Overview

The reports and plots in this directory provide a comprehensive overview of the project's methodology, results, and conclusions. They are the primary outputs of the experiments and are used to evaluate the performance of the recommendation model.
